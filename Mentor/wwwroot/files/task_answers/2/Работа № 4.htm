НЕЛИНЕЙНЫЕ ДИНАМИЧЕСКИЕ СТРУКТУРЫ. m-АРНЫЕ ДЕРЕВЬЯ.

Цель:
Изучение основных принципов построения динамических нелинейных структур и алгоритмов для их обработки.

Задание:
Реализовать программу в соответствии с семантикой варианта. Для операций с деревом реализовать следующие подпрограммы:
  • начальной инициализации (Создание корня дерева);
  • добавления к узлу нового потомка ;
  • удаления из узла заданного потомка;
  • поиска узла в дереве в соответствии с контекстом задания;
  • вывода дерева на экран.
Вывод должен быть реализован в соответствии с вариантом (префиксный, инфиксный, постфиксный).

Файле Metod4lab.doc


Содержание отчета:
Абстрактное представление структуры.
Представаление данных в графическом виде.
Проектирование необходимых классов и структур.
Алгоритм для каждого метода(класса/структуры), используя псевдокод или блок-схему.
Листинг программы обработки списка и экранные формы результа работы программы.
Примечание: Информация, хранимая в классе или структуре, должна содержать как минимум два поля
